# DeepLabV3+ 语义分割工作站 使用说明

> **产品定位**：Qt Widgets 工作站是唯一正式入口，覆盖数据、标注、训练、预测、评估和模型管理。QML Studio 是实验预览版，不承诺与正式入口功能对等。

## 启动

双击项目根目录的 **`启动工作站.bat`**，或在终端运行：

```
.venv\Scripts\python.exe run_workstation.py
# 英文界面（也可设置 RURALSCAPE_LANG=en_US）
.venv\Scripts\python.exe run_workstation.py --lang en_US
```

环境为项目内置虚拟环境 `.venv`（Python 3.12 + PyTorch 2.11 CUDA 12.8，已适配 RTX 5070 Ti）。默认语言为 `zh_CN`；支持 `--lang zh_CN|en_US`，环境变量 `RURALSCAPE_LANG` 的优先级低于启动参数。

实验 QML 入口：

```powershell
# connected：使用真实后端，仍显示实验预览定位
.venv\Scripts\python.exe run_studio.py
# demo：同一 StudioRuntime 契约，所有动作明确提示演示/不支持，不会静默
.venv\Scripts\python.exe -m ruralscape_studio
```

QML 界面持续显示实验水印；生产作业、完整标注与无障碍验收请使用 Widgets 工作站。

## 功能页

### 📁 数据管理
- 浏览 `VOCdevkit/VOC2007` 下所有 图片-标签 对，点击可预览 原图 / 着色标签 / 叠加图。
- **导入图片**：批量选择图片复制入库，可同时从某个文件夹按同名自动匹配标签 png。
- **导入标签**：单独补充标签（文件名需与图片一致）。
- **删除选中**：条目移入 `VOCdevkit/_trash/时间戳/`（可手动找回），并自动从划分中移除。
- **随机划分**：设置 训练+验证占比 / 训练占比 / 随机种子，重新生成 train/val/test.txt。
- 右键选中条目可手动**设为训练集/验证集/测试集**（即数据分类管理）。
- **检查数据集**：统计标签像素值分布，自动发现格式问题。
- **类别定义**：编辑类别名与显示颜色（第 0 类为背景），保存后全局生效；`num_classes = 类别数`。

### ✏️ 图像标注
- 左侧列表显示数据集全部图片（✓ 已有标签 / ○ 无标签），点击加载到画布。
- 工具：**画笔(B)** 涂抹当前类别、**橡皮(E)** 擦为背景、**多边形(P)** 单击加点后右键/双击闭合填充、
  **区域填充(F)** 将点击处的连通区域改为当前类别；`[` `]` 调笔刷大小，Ctrl+Z/Y 撤销重做。
- 视图：滚轮缩放、Shift+左键或中键拖动平移、“适应窗口”复位；标签不透明度可调。
- **🤖 AI 预标注**：用当前模型（沿用「图像预测」页的权值与参数）自动生成标签，再手动修正，效率最高。
- **SAM2 边界精修**（预标注下方复选框，默认开）：语义模型出类别、SAM2 修边界，
  高分辨率照片上边界质量明显更好；首次使用需联网下载模型（约 40MB）。
- **保存标签 (Ctrl+S)**：写入 `SegmentationClass/<名称>.png`（带调色板的索引 PNG，可直接用于训练）。
- **导出标签**：当前图片或全部已标注图片，支持 6 种格式——
  索引 PNG / 调色板 PNG / 彩色 PNG / 叠加 PNG / LabelMe JSON（多边形可再编辑）/ YOLO-seg TXT。

### 🎯 模型训练
- 左侧为全部关键参数：主干网络(mobilenet/xception)、下采样倍数、输入尺寸、初始权值、
  冻结/解冻轮次与 batch_size、优化器、学习率及衰减、Dice/Focal Loss、fp16、保存与评估周期等。
- 点击**开始训练**后在独立子进程中训练，右侧实时显示：Epoch/Step 进度条、
  训练/验证损失曲线、验证集 mIoU 曲线、日志控制台。
- **停止**：当前 epoch 结束后安全停止（已保存 last/best 权值）；**强制终止**立即杀进程。
- 权值保存到指定目录：`best_epoch_weights.pth`（按**验证集 mIoU** 挑选，
  关闭周期评估时退回按验证损失）、`last_epoch_weights.pth`、周期快照。
- 默认开启 **EMA**（指数滑动平均）：评估与最佳权重取 EMA 副本，周期快照保留原始权重供续训。

### 🖼️ 图像预测
- 选择权值（自动扫描 model_data/ 与所有 logs* 目录）、主干、输入尺寸、GPU/CPU。
- 主干支持两套架构：`mobilenet / xception`（DeepLabV3+ 权重）与
  `segformer-b0 / b1 / b2`（`tools/train_segformer.py` 训练的权重），必须与权重文件匹配。
- **高质量模式 (TTA)**：翻转×多尺度概率平均，DeepLab 权重约 +0.7 mIoU，
  单张耗时约 6 倍；批量/视频建议关闭。
- **单张预测**：原图与结果并排显示，附每个类别的像素数与占比统计；
  显示方式可切换 叠加混合 / 仅分割图 / 扣除背景，透明度可调（无需重新推理）。
- **批量预测**：整个文件夹推理并保存，可选同时输出原始类别索引 mask。

### 📊 精度评估
- 选择权值与评估划分（val/test/train），一键计算 mIoU / mPA / Accuracy 及
  每类 IoU、Recall、Precision 表格；结果图与混淆矩阵保存到 `miou_out/`。
- 口径说明：指标在**完整混淆矩阵**上计算，背景类展示但不计入 mIoU/mPA 平均；
  与 `get_miou.py`、训练中的周期评估完全一致。支持两套架构（同预测页）。

### 💾 模型管理
- 列出所有权值文件（大小/时间），支持导入、重命名、打开所在文件夹。删除会移入 `models/_trash/<时间戳>/`，可通过页内提示撤销最近一次删除。

## 注意事项
- 修改类别数后，旧权值与新配置不匹配，需要重新训练；预测时的 主干/下采样/类别数 必须与训练时一致。
- 数据集图片与标签均为 png，标签像素值 = 类别序号（0 为背景）。
- 所有参数修改会自动保存到 `workstation_config.json`。
