# RuralScapes 键盘与读屏基线

Widgets 工作站是唯一正式入口；QML 为实验预览版。

## 键盘

- Ctrl+1～Ctrl+6：切换数据、标注、训练、预测、评估和模型页。
- 上/下方向键：在侧栏循环移动并激活页面。
- F6：在侧栏和当前内容区之间切换焦点。
- Ctrl+S / Ctrl+Z / Ctrl+Y：标注保存、撤销、重做。
- B / E / P / F、[ / ]：仅在标注画布拥有焦点时切换工具或笔刷大小。

## 读屏走查

Windows Narrator 或 NVDA 应读出导航名称、按钮角色、禁用状态、输入框提示、
配置错误字段和标注画布说明。标注画布描述包含当前工具和未保存状态；SVG 图标
只作装饰，不进入可访问名称。

正式发布前在真实 Windows 桌面依次完成：选择数据集、进入训练配置、触发启动前
校验、打开标注页、保存标签、将模型移入回收站并撤销。离屏自动测试不能替代这次
人工走查。
